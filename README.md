Portafolio web personal desarrollado con HTML, Tailwind CSS y JavaScript vanilla. Diseñado para presentar mi perfil como **Desarrollador Backend Junior**, mis proyectos y mi experiencia profesional.

[Ver portafolio](https://alancontreras.vercel.app/)

- **Sección Sobre Mí** — Presentación personal, ubicación y links de contacto
- **Tecnologías** — Stack backend, bases de datos, frontend e infra/DevOps
- **Trayectoria Profesional** — Experiencia en desarrollo y soporte técnico
- **Proyectos Destacados** — API de Autenticación JWT y Vault API

## Cómo correrlo localmente

No requiere instalación de dependencias. Solo abre el archivo directamente en tu navegador:

```bash
# Opción 1: Abrir directo
start portafolio/index.html

# Opción 2: Con Live Server (extensión de VS Code recomendada)
# Clic derecho en index.html → "Open with Live Server"
```
